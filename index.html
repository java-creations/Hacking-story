1. What is Kali Linux?

Kali Linux is a Debian-based Linux distribution used by ethical hackers and penetration testers. It comes pre-installed with hundreds of security tools for testing network vulnerabilities, web applications, wireless networks, and more.

2. Setting Up Kali Linux

You can install Kali Linux on bare metal, in a virtual machine, or use a live USB. Recommended for beginners: VirtualBox or VMware.

# Download Kali ISO https://www.kali.org/get-kali/ # Update system after install sudo apt update && sudo apt upgrade # Install basic tools sudo apt install net-tools git curl build-essential
